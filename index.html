backbone-fill-collection

Backbone-Fill-Collection makes it easy to generate collections with mock data quickly for prototyping. It sends requests to filltext.com and inserts the results into a backbone collection on every fetch.

Download

download master/edge

Usage

Extend a collection from Backbone.FillCollection and insert a fillWith property with a hash of parameters. The key will become the models key and it's value can be static or mocked with the FillText params. A full list of FillText's params can be found on their website.

User = Backbone.Model.extend({
  defaults: {
    name: 'n/a',
    age: 'n/a',
    homeTown: 'n/a',
    email: 'n/a'
  }
});

Users = Backbone.FillCollection.extend({
  model: User,

  fillWith: {
    // optional number of models to return, defaults to 5
    rows: 3,

    name: '{firstName}',
    homeTown: '{city}',
    email: '{email}',
    type: 'user'
  }
});

Outputs 

users = new app.Users();
users.fetch();
